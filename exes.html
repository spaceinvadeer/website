How to run exe's

1. press windows key+r

2. type in: C:\Windows\SysWOW64

3. find the folder called Tasks

4. create a new folder anywhere

5. create a shortcut for the new folder by right clicking it and clicking show more options

6. drag the folder not the shortcut into Tasks

7. use the shortcut to get to tasks

8. anything inside tasks will run
